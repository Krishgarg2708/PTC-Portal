# 📌 Overview

This project is a lightweight frontend application designed to simplify spreadsheet distribution within an organization.

Instead of sharing raw Google Sheets links, users are presented with a clean interface where they can:

* Download the latest spreadsheet
* Read usage instructions
* View access permissions
* Understand color-based editing rules
* Switch between English and Hindi

The application is intentionally minimal while maintaining a modern and professional UI.

# 🚀 Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Move into the project

```bash
cd power-technologies-download-portal
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
